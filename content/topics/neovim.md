# What is neovim
[Neovim](https://neovim.io/) is a modal text editor, born as a fork of [vim](https://www.vim.org/). A modal editor differs from the popularized modeless editor model which can be seen in apps like Window's Notepad. 


I have been using Neovim for a few years now, and the experience has been mostly great. With one minor caveat: reproducibility. This is now solved thanks to Nix. Trying to improve my setup, you can read about the efforts in [[nixvim]].
